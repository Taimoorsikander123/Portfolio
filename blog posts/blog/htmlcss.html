HTML and CSS

HTML

HTML (Hypertext Markup Language) was first developed by Tim Berners in 1990. HTML 1.0 was the first version published in 1993 and was updated throughout the years. Latest version is HTML 5 published in 1993. Every webpage is created by using HTML. Each webpage is connected by hyperlinks. Text, images and other content are presented in desired format. HTML file is created to develop web pages. HTML files contain plain text with HTML tags and content required for a web page.

HTML Tags

“HTML tags are hidden keywords within a webpage that define how your browser must format and display the content.” HTML tags mostly have two part 1) Opening tag 2) Closing tag For example, <div>gt; is the opening tag and </div> is the closing tag. But some HTML tags don’t have a closing tag. For example, has only an opening tag.

HTML Elements

Start tags and closing tags with content inside them make HTML Elements.
For example, div&<p> This is paragraph </p>
This p element defines a paragraph.
HTMl attributes describe extra information about elements. HTML attributes are in the opening tag.

Inline and Block Elements

Inline elements occupy required space and are adjusted on the same line. For example, <i>, <img>, <button>, and dozens of more. Block elements occupy the full width available. Block elements always establish on new lines. Block elements have margin on top and bottom. For example, <div>, <li>, <p>, and many more.

CSS

CSS ( Cascading Style Sheets) is language used to present a web page in terms of colors, fonts and styles. CSS was first developed by Hakon Wium Lie in 1994 and published in 1996.

CSS Selector And Declaration Block

A CSS value has a selector and declaration block. CSS selector targets specific HTML elements to design. A Declaration Block has one or more declaration properties that are distinguished by semicolons . All declaration properties of a specific selector are inside curly brackets. For example,
p { color: blue; text-align: center }
